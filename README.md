# 🎓 Simulado AB-900 – Microsoft 365 Copilot & Agent Administration Fundamentals

Este projeto é um **simulador interativo em formato web** desenvolvido para apoiar os estudos da certificação  
**Microsoft 365 Copilot & Agent Administration Fundamentals (AB-900)**.

O objetivo é oferecer uma experiência prática de estudo, combinando **modo simulado (com explicações)** e  
**modo prova (com tempo e pontuação no padrão Microsoft)**.

---

## 🧠 Funcionalidades

### 📘 Modo Simulado
- Apresenta **as 50 questões** uma a uma
- Permite visualizar:
  - ✅ Resposta correta/incorreta
  - 📖 Explicação detalhada
- Ideal para estudo e fixação dos conceitos

### 📝 Modo Prova
- 50 questões sorteadas aleatoriamente
- ⏱️ Tempo cronometrado
- 🧮 Pontuação no **padrão Microsoft**:
  - Cada questão vale **20 pontos**
  - Total máximo: **1000 pontos**
  - ✅ **Aprovado** com **≥ 700 pontos**
- Resultado exibido ao final em **pop-up/modal**

### 🎯 Outras características
- Questões **embaralhadas** a cada tentativa
- Interface leve, responsiva e intuitiva
- Funciona diretamente no navegador (sem backend)

---

## 📚 Conteúdo abordado

As questões foram baseadas no **Study Guide oficial da certificação AB-900**, cobrindo temas como:

- Fundamentos do Microsoft 365
- Identidade e acesso (Microsoft Entra ID)
- Zero Trust, MFA e Conditional Access
- Governança e proteção de dados (Microsoft Purview)
- Oversharing e riscos com Copilot
- Auditoria, DLP e retenção
- Administração básica do Copilot
- Agentes e boas práticas de governança
